# CB Stats

Stats aggregation and prediction for Coding Blocks.

```bash
skaffold dev --port-forward
```

Open the following sites in a browser:

- [Grafana](http://localhost:80) (admin/prom-operator)
- [Prometheus](http://localhost:9090)
